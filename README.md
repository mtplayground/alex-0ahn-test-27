# alex-0ahn-test-27

Vite + TypeScript project scaffold for the fluid playground application.

## Requirements

- Node.js 20+
- pnpm 10+

## Development

```bash
pnpm install
pnpm dev
pnpm lint
pnpm format
```

The Vite dev server is configured to listen on `0.0.0.0:8080`.

## Build

```bash
pnpm build
pnpm preview
```

## Environment

Copy `.env.example` to `.env.local` to override local settings.
